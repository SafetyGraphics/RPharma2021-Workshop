R/Pharma safetyGraphics Workshop

ASA Biopharm-DIA Safety Working Group

October 29, 2021

Materials for the safetyGraphics R Pharma 2021 Workshop on 2021-10-29 from 9 AM - 12 noon ET.

The Interactive Safety Graphics (ISG) subteam of the ASA Biopharm-DIA Safety Working Group is an interdisciplinary effort that, amongst other things, seeks to provide a clinical safety workflow for monitoring during clinical development in an open source model.

Sign up here if you’d like to learn more or get involved!

Setup

We’ll be using RStudio Cloud for this workshop. Click this link to an RStudio Project that includes all examples, exercises and Slides (pptx).

⚠️ Caution: Since we’re using a cloud-based platform, we do not recommend using any private data during the workshop. Sample data is provided for all examples. ⚠️
